Building OSPF Network with FRRouting and Docker

This is a lab project in which I built an OSPF network among Docker containers, then performed network experiments and package analysis. Testings are performed under Virtual Box in Ubuntu 16.04 system.

In the experiments, the basic network topology is shown as follows:

Our experiments involves:
  • Add costs to each link and observe changes in routing table
  • Send packets using iperf capture using tcpdump
  • Break certain links, analyze the packets using Wireshark
